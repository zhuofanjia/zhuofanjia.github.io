# Zhuofan Jia - Personal Academic Website

This repository hosts a static personal website for GitHub Pages.

## Structure

- `index.html`: Main homepage content.
- `stylesheet.css`: Styling and responsive layout.
- `assets/images/profile.jpg`: Profile photo.
- `assets/images/research-graph.png`: Placeholder research graph image.
- `resume.tex`: Source resume in TeX format.

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. In GitHub, open **Settings -> Pages**.
3. Under **Build and deployment**, choose:
	- **Source**: Deploy from a branch
	- **Branch**: `main` (or your default branch), `/ (root)`
4. Save and wait for deployment.

Because this site is fully static and uses relative paths, it is ready to deploy directly from the root folder.
